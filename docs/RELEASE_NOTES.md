# Release Notes

## December 2, 2025

### New Features

#### Admin Premium Type Management

**What's new:**
Admins can now manage premium types for user team plans directly from the settings page.

**What you'll notice:**
- A new "Manage Premium Type" menu item appears in settings for admin users
- Admins can search and select users by email address or name to assign premium access
- Premium types can be updated for any user's team plan
- View a complete list of all current premium users
- Quickly reset users back to free tier with one click
- See total count of premium users at a glance

### Bug Fixes

#### Metric Diagram Display Issue (JIRA-1250)

**What was fixed:**
Linked dashboards and reports now display correctly in the metric diagram view.

**What you'll notice:**
- When viewing a metric's diagram, all associated dashboards and reports will now appear properly
- Visual connections between metrics and their related dashboards/reports are now accurate
- You can now see the complete relationship map for each metric

#### GoJS License Key Error in Metric Offcanvas

**What was fixed:**
GoJS license key is now loaded in the correct order, preventing JavaScript errors in the metric offcanvas component.

**What you'll notice:**
- Metric diagrams in offcanvas views now load without errors
- Improved stability when viewing metric relationships
